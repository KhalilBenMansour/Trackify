## Installing ðŸ§°
1. Clone the Repository
```
  git clone https://github.com/KhalilBenMansour/Trackify.git 
  cd trackify
```
1. Install dependencies
```
  npm i && cd client npm i
```
1. Create .env file
```
  MONGO_URI="your mongoDb atlas"
  PORT=5000
  secretOrKey="your secret key"
```
1. Run the Project
```
  npm run dev
```
